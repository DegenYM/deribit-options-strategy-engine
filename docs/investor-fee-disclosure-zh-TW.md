# 投資人分潤與計費說明書

**Deribit 期權策略組合**｜版本 1.2｜績效費 10%｜管理費年化 1.0%

> 正式權利義務以雙方簽署之投資管理協議為準。

---

## 一、費用結構總覽

| 項目 | 費率／原則 | 說明 |
|------|------------|------|
| 管理費 | 年化 1.0%（按季結算） | 按 AUM_mgmt 季度平均規模計提 |
| 績效費 | 10% | 僅就超過 HWM 之新創利潤抽取 |
| 計費單元 | 投資人合併 | 所有**策略**授權子帳合併計算 NAV |
| 高水位 HWM | 必備 | **僅綁定 NAV_perf** |
| 費用收取 | **季末賣現貨兌穩定幣** + **外部鏈上付費** | 管理方在策略子帳 Trade 兌 USDC／USDT／USDE；投資人確認帳單後轉至管理方指定外部地址 |

---

## 二、兩套計費基礎

### NAV_perf（績效 NAV）— 不含備兌現貨

- **定義**：各**策略**子帳 USDC 等價總權益 − 約定備兌現貨市值  
- **用途**：期間損益、HWM、10% 績效費  
- **不計入**：投資人自有 BTC/ETH 備兌現貨及其漲跌  

### AUM_mgmt（管理費資產）— 含備兌現貨

- **定義**：NAV_perf + 備兌現貨市值  
- **用途**：1% 管理費（現貨佔保證金與託管資源）  

---

## 三、多幣種換算

結算日 23:59 UTC，各**策略**子帳 book 權益 × Deribit 指數價 → USDC 等價。

```
NAV_perf = Σ 策略子帳 USDC 等價權益 − 備兌現貨 USDC 等價
AUM_mgmt = NAV_perf + 備兌現貨 USDC 等價
```

---

## 四、績效費與 HWM

```
可分配利潤 = max(0, NAV_perf,期末 − HWM − 淨申赎調整)
績效費 = 可分配利潤 × 10%
HWM 僅追蹤 NAV_perf，不含現貨
```

---

## 五、費用收取：季末兌幣 + 外部地址付費

管理費與績效費**不**自策略子帳內直接扣款，改採下列方式：

### 5.1 管理方（結算後）

1. **帳單**：結算日後出具 USDC 等價帳單（含管理費、績效費明細）。  
2. **賣現貨（若需要）**：若有獲利現貨需兌付費用，管理方在**策略子帳**以 API **Trade** 將對應部分賣成 **USDC、USDT 或 USDE**（穩定幣留在該子帳，供雙方在 Deribit 後台對照）。  
3. **對帳**：與投資人確認帳單金額；必要時核對子帳內穩定幣餘額與帳單一致。

管理方**不**要求主帳 API，也**不**執行 Deribit 子帳之間的 Internal transfer（該操作需主帳授權）。

### 5.2 投資人（付費）

1. 收到帳單並與管理方完成對帳確認。  
2. 將應付金額轉至管理方指定的**外部鏈上地址**（見 `config/platform/fee-payout-addresses.toml`；簽約時會提供正式清單）。  
3. 常見幣別：**USDC、USDT、USDE**；**鏈／網路**須與管理方指示一致（多為 Ethereum ERC-20）。  
4. 可從 Deribit **主帳 Withdraw** 提至該地址，或從自有交易所／錢包直接轉帳。

### 5.3 策略子帳 API 權限

| Deribit scope | 策略子帳 |
|---------------|----------|
| account:read | ✅ 要開 |
| trade:read_write | ✅ 要開（期權 + 季末賣現貨兌穩定幣） |
| wallet | ❌ **none**（不開；管理方無主帳 API） |

### 5.4 幣別與逾期

- 帳單以結算日 **USDC 等價** 開立；鏈上付費建議使用帳單指定之 **USDC、USDT 或 USDE**，避免換幣爭議。  
- 逾期未確認對帳或未完成鏈上付費，管理方得依投資管理協議暫停新開倉，直至結清。

## 六、結算週期（摘要）

| 項目 | 建議約定 |
|------|----------|
| 定期結算 | 每季末（UTC） |
| 帳單出具 | 結算日後約 10 個工作天內 |
| 投資人付費 | 對帳確認後約 5 個工作天內，轉至管理方外部地址 |
| 贖回 | 贖回生效日另結算應計費用 |

## 附錄：HWM 是否包含現貨？

**不包含。** HWM 只跟 NAV_perf；現貨跌不直接拉低高水位。管理費無 HWM，按含現貨的 AUM_mgmt 計提。
