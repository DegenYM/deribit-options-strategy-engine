# 投資人分潤與計費說明書

**Deribit 期權策略組合**｜版本 1.1｜績效費 10%｜管理費年化 1.0%

> 正式權利義務以雙方簽署之投資管理協議為準。

---

## 一、費用結構總覽

| 項目 | 費率／原則 | 說明 |
|------|------------|------|
| 管理費 | 年化 1.0%（按季結算） | 按 AUM_mgmt 季度平均規模計提 |
| 績效費 | 10% | 僅就超過 HWM 之新創利潤抽取 |
| 計費單元 | 投資人合併 | 所有**策略**授權子帳合併計算 NAV |
| 高水位 HWM | 必備 | **僅綁定 NAV_perf** |
| 費用收取 | 獨立 **Fee 專戶** + **主帳提幣** | 管理方換幣劃轉至 Fee 專戶對帳；投資人確認後自主帳提至管理方指定地址 |

---

## 二、兩套計費基礎

### NAV_perf（績效 NAV）— 不含備兌現貨

- **定義**：各**策略**子帳 USDC 等價總權益 − 約定備兌現貨市值  
- **用途**：期間損益、HWM、10% 績效費  
- **不計入**：投資人自有 BTC/ETH 備兌現貨及其漲跌  
- **不含** Fee 專戶餘額（費用專戶不參與 NAV_perf／HWM）

### AUM_mgmt（管理費資產）— 含備兌現貨

- **定義**：NAV_perf + 備兌現貨市值  
- **用途**：1% 管理費（現貨佔保證金與託管資源）  

---

## 三、多幣種換算

結算日 23:59 UTC，各**策略**子帳 book 權益 × Deribit 指數價 → USDC 等價。

```
NAV_perf = Σ 策略子帳 USDC 等價權益 − 備兌現貨 USDC 等價
AUM_mgmt = NAV_perf + 備兌現貨 USDC 等價
```

---

## 四、績效費與 HWM

```
可分配利潤 = max(0, NAV_perf,期末 − HWM − 淨申赎調整)
績效費 = 可分配利潤 × 10%
HWM 僅追蹤 NAV_perf，不含現貨
```

---

## 五、費用收取：Fee 專戶對帳 + 主帳提幣

管理費與績效費**不**自策略子帳內直接扣款，改採下列方式：

### 5.1 Fee 專戶是什麼？

- 投資人在 Deribit 主帳下另開一個**獨立子帳戶**（建議名稱：`fee_acc`；至少 5 字元，勿用 `fee`），作為季結算後的**對帳專戶**。  
- **策略程式只連策略子帳**（如 `naked`、`covered_call`）；**不會**操作 Fee 專戶下單。  
- Fee 專戶餘額不計入 NAV_perf／HWM。

### 5.2 季結算收取流程

1. **帳單**：結算日後管理方出具 USDC 等價帳單（含管理費、績效費明細）。  
2. **換幣與劃轉（管理方）**：若有獲利現貨需兌付費用，管理方先在策略子帳將對應部分 **TRADE 成 USDC 或 USDT**，再以策略子帳 API（Wallet=read_write）**Internal transfer** 劃至 Fee 專戶。  
3. **對帳（雙方）**：管理方透過 Fee 專戶 API（Account=read）查餘額；投資人在 Deribit 後台核對 Fee 專戶餘額與帳單一致。  
4. **付費（投資人）**：確認無誤後，投資人從**主帳**以 **Withdraw** 提幣至管理方指定的鏈上地址（幣別、鏈、金額以帳單與管理方指示為準）。

### 5.3 投資人要做的事

1. 建立 Fee 專戶子帳（與策略子帳分開）。  
2. 為 Fee 專戶建立**專用 API Key**（見 5.4），透過安全管道交給管理方（供對帳查餘額用）。  
3. 收到帳單後，與管理方完成對帳並書面／訊息確認。  
4. 自**主帳**提幣至管理方指定地址，完成付費。  
5. **策略子帳**的 API Key 需開 **Wallet=read_write**（供管理方 API 劃轉至 Fee 專戶）；**主帳** API **勿**交付管理方。

### 5.4 API Key 權限（策略子帳 vs Fee 專戶）

| Deribit scope | 策略子帳 | Fee 專戶 |
|---------------|----------|----------|
| account:read | ✅ 要開 | ✅ 要開 |
| trade:read_write | ✅ 要開 | ❌ 不開 |
| wallet:read_write | ✅ **要開**（API 劃轉至 Fee 專戶） | ❌ **不要開** |

管理方使用**策略子帳** API 的 Wallet 權限做內部劃轉；**Fee 專戶** API 僅 Account=read 供對帳查餘額。**實際收取**由投資人自**主帳**提幣完成。

### 5.5 幣別與逾期

- 帳單以結算日 **USDC 等價** 開立；對帳專戶建議以 **USDC 或 USDT** 存放，避免換幣爭議。  
- 逾期未確認對帳或未自主帳完成付費，管理方得依投資管理協議暫停新開倉，直至結清。

## 六、結算週期（摘要）

| 項目 | 建議約定 |
|------|----------|
| 定期結算 | 每季末（UTC） |
| 帳單出具 | 結算日後約 10 個工作天內 |
| 投資人付費 | 對帳確認後約 5 個工作天內，自**主帳**提至管理方指定地址 |
| 贖回 | 贖回生效日另結算應計費用 |

## 附錄：HWM 是否包含現貨？

**不包含。** HWM 只跟 NAV_perf；現貨跌不直接拉低高水位。管理費無 HWM，按含現貨的 AUM_mgmt 計提。
